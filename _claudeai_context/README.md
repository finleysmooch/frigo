# _claudeai_context/

Short-lived code/file snapshots staged by Claude Code for a specific Claude.ai session to consume via drag-drop chat attachment.

**This folder is gitignored.** Contents are ephemeral — staged at the start of a session, cleared when the session ends.

**Distinct from `_pk_sync/`:** `_pk_sync/` stages living-doc edits for durable PK replacement. This folder stages code/file snapshots for one-session context.

See `docs/DOC_MAINTENANCE_PROCESS.md` for the full convention.
