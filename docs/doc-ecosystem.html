Weekly Sync — What Gets Updated
1. Gather unprocessed SESSION_LOG entries from repo 5 min 2. Reconcile into active phase doc (progress, decisions, deferred items) 10 min 3. Act on recommended doc updates — update ARCHITECTURE + PROJECT_CONTEXT as needed 10 min 4. Upload updated docs to project knowledge, remove stale files 5 min 5. Update FRIGO_TRACKER ↗ with committed code rows 5 min 6. Update Claude Shared Reference ↗ if DB schema changed if needed 7. Push ARCHITECTURE + DEFERRED_WORK to repo if changed 2 min 8. Clear processed SESSION_LOG entries 1 min
Total: ~30 min with Claude
