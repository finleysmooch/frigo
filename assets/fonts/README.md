This directory contains the following fonts:

### Poppins (assets/fonts/Poppins/)

- Poppins-Regular.ttf (400)
- Poppins-Medium.ttf (500)
- Poppins-SemiBold.ttf (600)
- Poppins-Bold.ttf (700)

### Outfit (assets/fonts/Outfit/static/)
Downloaded from: https://fonts.google.com/specimen/Outfit
- Outfit-Regular.ttf (400)
- Outfit-Medium.ttf (500)
- Outfit-SemiBold.ttf (600)
- Outfit-Bold.ttf (700)

The logo component will automatically use these fonts when you select Poppins or Outfit as the font family in the Logo Playground.
